# Maa Vindhawasini Enterprises — Asset Pack

This ZIP was assembled from the approved modern UI concept boards created in the conversation.

## Folders

- `source-mockups/` — full concept boards for Home, About, Products, Product Detail, Custom Fabrication, Projects, Quality, Contact, and Request a Quote.
- `images/` — raster assets cropped from those boards.
- `icons/general/` — reusable custom line icons in SVG.
- `icons/products/` — seven product-category SVG icons.
- `design.md` — complete visual design and implementation guidance.
- `asset-manifest.json` — source and crop metadata for every extracted raster asset.

## Production warning

The original design generations are flattened raster images, so this pack cannot contain true layered Figma assets, original stock photos, or vector versions of the generated logos. The extracted images are suitable for development placeholders and design reference. Replace them with original client photographs and a professionally vectorized logo before production launch.
